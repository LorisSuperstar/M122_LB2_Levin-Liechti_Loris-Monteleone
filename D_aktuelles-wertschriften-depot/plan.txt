# Plan: D - Aktuelles Wertschriften-Depot

This project implements a fictitious investment portfolio tracker in Bash.
It tracks:
- 10 shares of Novartis (NOVN.SW)
- 3000 USD
- 0.1 Bitcoin (BTC)

And calculates historical vs. current value in Swiss Francs (CHF).

## Implementation Steps

1. **Setup & Config**:
   - Create a configuration file (`depot.cfg`) to store assets (asset amounts, purchase prices in CHF, and ticker names).
   - Create a history file (`depot_history.csv`) and log file (`depot.log`) to store run logs and historical values.

2. **Retrieve Current Rates (APIs)**:
   - BTC-CHF price: Fetch from Coinbase API (`https://api.coinbase.com/v2/prices/BTC-CHF/spot`).
   - USD-CHF rate: Fetch from Coinbase API (`https://api.coinbase.com/v2/prices/USD-CHF/spot`).
   - Novartis (NOVN.SW) price in CHF: Fetch from Yahoo Finance chart API (`https://query1.finance.yahoo.com/v8/finance/chart/NOVN.SW?interval=1d&range=1d`).

3. **Data Parsing (Bash with Python3 JSON parsing helper)**:
   - Extract numerical prices from raw JSON responses using safe Python3 inline json parsing (since `jq` is not installed on the system).
   - Validate that all retrieved prices are valid numbers.

4. **Calculate Depot Value**:
   - Retrieve historical cost of assets (e.g. initial CHF investments) from config or define constants.
   - Calculate current values of each asset class (Stock, Cash, Crypto) in CHF.
   - Sum them up to get the total current depot value.
   - Calculate total profit/loss (current value - historical cost).

5. **Historical Zeitschreibung (Time logging)**:
   - Format: Date, Time, Stock_Val, Cash_Val, Crypto_Val, Total_Current_Val, Profit_Loss.
   - Append this data to `depot_history.csv`.
   - Log execution status, timestamp, and errors to `depot.log`.

6. **Automation**:
   - Provide a way to run the script via standard automated triggers (like crontab).
